Мавка 0.113.0

Мавка 0.113.0 це найбільш актуальна версія Мавки, написана мовою Ціль.

Мавка 0.113.0 все ще активно допрацьовується.

GNU/Linux x86_64

Щоб використовувати Мавку 0.113.0 в операційних системах на основі Linux, вам необхідно збудувати її власноруч. Існує два способи.

Швидкий спосіб

Цей спосіб потребує лише встановленого Clang та Git.

Клонуємо репозиторій з підготовленими файлами:

git clone https://github.com/mavka-ukr-static/mavka-linux-x86_64 --depth=1

Відкриваємо клоновану теку:

cd mavka-linux-x86_64

Будуємо:

sh build.sh

Пробуємо:

./out/мавка

Спосіб для розробників

Цей спосіб вам знадобиться, якщо ви хочете вносити зміни в код. Окрім Clang та Git цей спосіб також потребує Цілі.

Клонуємо репозиторій Мавки:

git clone https://github.com/mavka-ukr/mavka --depth=1

Відкриваємо клоновану теку:

cd mavka

Компілюємо файли Цілі:

TSIL="ціль" sh external/scripts/prepare.sh linux-x86_64

Переходимо в підготовлену теку:

cd build/linux-x86_64

Будуємо виконуваний файл:

sh build.sh

Пробуємо:

./out/мавка

Windows

Файли для компіляції Мавки 0.113.0 на Windows x86_64, а також автоматично cross-скомпільований exe-файл, можна отримати тут: https://github.com/mavka-ukr-static/mavka-windows-x86_64.
